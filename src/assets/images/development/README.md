# Development Block Images

Place 3 images here for the Development block carousel:
- `1.jpg` - First image
- `2.jpg` - Second image  
- `3.jpg` - Third image

Each image will be displayed in the carousel next to the Development block.

Recommended:
- Format: JPG, PNG, or WebP
- Size: ~800-1200px width
- Aspect ratio: Landscape (16:9 or 4:3)
